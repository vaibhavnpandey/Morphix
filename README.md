# Morphix
Swarm Robotics Research Platform

A modular swarm robotics project focused on designing and developing autonomous robots capable of cooperative behavior and distributed task execution.

This repository documents the development of a swarm robotics platform consisting of a Master Robot and multiple Slave Robots. The primary objective is to build a scalable system for research in autonomous navigation, multi-agent coordination, and swarm intelligence.

At the current stage, the project focuses on the hardware design of the robots. Firmware, communication protocols, and autonomous algorithms will be added in future updates.
